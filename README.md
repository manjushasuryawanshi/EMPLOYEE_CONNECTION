# Employee Management System

## 📌 About the Project

Employee Management System is a Full Stack MERN application that performs CRUD (Create, Read, Update, and Delete) operations on employee records.

The application is built using React.js for the frontend, Node.js and Express.js for the backend, and MongoDB as the database.

---

## ✨ Features

- ➕ Add Employee
- 📋 View Employee Details
- ✏️ Update Employee Information
- 🗑️ Delete Employee
- 📡 REST API Integration
- 💾 MongoDB Database
- 📱 Responsive Interface

---

## 🛠️ Technologies Used

- React.js
- Node.js
- Express.js
- MongoDB
- Mongoose
- Axios
- CSS

## 🚀 How to Run

### Backend

```bash
cd backend
npm install
node server.js
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
